What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing the structure as a tree of objects. It allows scripts to dynamically access and manipulate HTML or XML content.

How do you select an element from the DOM?

To select an element from the DOM, use JavaScript methods like getElementById, querySelector, or getElementsByClassName to target elements by ID, CSS selectors, or class names for manipulation.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events from its children. It improves performance and simplifies dynamic content handling.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM, use setAttribute() or getAttribute() for attributes, and modify the style property for CSS, like element.style.color = "red";.
